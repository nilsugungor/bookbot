# BookBot

**BookBot** is a simple Python command-line tool that analyzes the text of a book provided as a file path. It performs the following tasks:

- Counts the total number of words in the book.
- Calculates the frequency of each character in the text.
- Sorts the characters by frequency and displays them in a clear report.

## Usage

Run the script with the path to your text file as an argument:

```bash
python3 main.py <path_to_book>
